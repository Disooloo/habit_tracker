# Мягкий трекер привычек

Мобильное приложение для формирования привычек без давления, ориентированное на людей с прокрастинацией и проблемами фокуса (в т.ч. ADHD).

## Особенности

- **Микродействия** вместо больших целей
- **Отсутствие стыда** и «срывов серий»
- **Поддержка попыток**, а не только успехов
- **Быстрый старт** - 30 секунд для начала
- **Мягкие напоминания** с позитивными формулировками

## Технологии

- Flutter (latest stable)
- Clean Architecture
- Bloc для state management
- SQLite для локального хранения
- Локализация (RU/EN)

## Установка

```bash
flutter pub get
flutter run
```

## Структура проекта

```
lib/
├── main.dart
├── core/           # Константы, тема, утилиты, локализация

├── domain/         # Entities, Use Cases
├── presentation/   # Bloc, UI (экраны и виджеты)
└── services/       # Уведомления, подписка
```
